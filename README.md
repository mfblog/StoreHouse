### MyBox 代理服务控制中心

MyBox 是一个现代化的代理服务控制中心，专为管理 Sing-Box 和 MosDNS 服务而设计。提供直观的 Web 界面，支持配置管理、服务控制和实时监控。

#### ✅ 已完成功能

##### 🎯 核心功能
- **服务管理**: 启动、停止、重启 Sing-Box 和 MosDNS 服务
- **状态监控**: 实时显示服务运行状态、CPU、内存使用情况
- **版本信息**: 自动获取并显示核心版本和分支信息
- **日志查看**: 实时查看服务日志，支持自定义行数

##### 🚀 Sing-Box 管理
- **配置管理**: 完整的 Sing-Box 配置文件编辑和验证
- **入站配置**: 支持 HTTP、SOCKS、Hysteria2、TProxy 等协议
- **代理节点**: 可视化管理代理节点，支持多种协议
- **出站配置**: 灵活的出站规则配置和管理
- **路由规则**: 智能路由规则编辑，支持域名、IP、GeoSite 分流
- **DNS 配置**: DNS 服务器配置和分流规则管理
- **网络状态**: 显示路由表和防火墙规则

##### 🌐 MosDNS 管理
- **服务控制**: MosDNS 服务的启动、停止、重启
- **状态监控**: 实时监控 MosDNS 运行状态
- **配置查看**: 查看 MosDNS 配置信息
- **版本显示**: 显示 MosDNS 核心版本信息

##### 🎨 界面特色
- **响应式设计**: 完美适配桌面端和移动端
- **现代化 UI**: 使用 Vue 3 构建，界面美观现代
- **实时更新**: 自动刷新服务状态和系统信息
- **操作反馈**: 完善的操作成功/失败提示
- **配置备份**: 自动备份配置文件，支持一键恢复

##### 🔧 技术特性
- **轻量级**: 单一二进制文件，无外部依赖
- **跨平台**: 支持 Linux AMD64 和 ARM64 架构
- **嵌入式**: Web 界面完全嵌入，无需额外文件
- **API 接口**: 提供完整的 RESTful API
- **配置验证**: 实时配置文件语法验证
